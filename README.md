## 🎓 Sobre o Projeto

O **Brito Core Systems | Academic Dash** é uma aplicação web desenvolvida como **projeto acadêmico da Universidade Estácio de Sá**, no curso de **Análise e Desenvolvimento de Sistemas (ADS)**.  

O sistema foi criado para auxiliar estudantes no **registro e monitoramento de notas acadêmicas**, permitindo o cálculo automatizado de médias e a visualização gráfica do progresso escolar.  

O projeto reflete a metodologia de *Elite Engineering*, com foco em **interfaces minimalistas**, **alta performance** e **controle preciso sobre o ciclo de vida dos dados**.

## ✨ Funcionalidades

### 📝 Gestão Acadêmica
* Cadastro de perfil (Nome, Curso, Semestre e ID)  
* Registro individualizado de notas por disciplina  
* Interface intuitiva para entrada e remoção de dados  

### 📊 Análise de Desempenho
* Cálculo automático de médias aritméticas  
* Gráfico de barras dinâmico para comparação entre matérias  

### 💾 Persistência de Dados
* Salvamento em tempo real no navegador  
* Recuperação automática de histórico ao acessar o sistema  

### 🎨 Interface
* Estética *Industrial Minimalist*  
* Layout responsivo e otimizado para diferentes dispositivos

## 📖 Como Utilizar

1. Preencha o formulário com seus dados acadêmicos.

2. Digite o nome da disciplina e a nota obtida.

3. Clique em Adicionar Nota.

4. Visualize o gráfico atualizado automaticamente.

5- Utilize Resetar para limpar os dados ao final do semestre.
